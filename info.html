Полезная информация

Данный материал скачан с портала PHP.SU.

Портал PHP.SU посвящен наиболее популярному языку веб-программирования PHP и веб-технологиям, которые имеют прямое отношение к PHP, таким как MySQL.

Портал PHP.SU содержит большое количество полезной информации по веб-технологиям, актуальной как для начинающих веб-программистов, так и для продвинутых.

На данный момент PHP.SU является одним из крупнейших специализированных сайтов.

На нашем портале доступны материалы для изучения PHP, справочник по функциям PHP на русском языке (свыше 4000 функций), сборник различных статей, PHP скрипты (более 2000), полезные программы для работы с PHP и MySQL, форум и многое другое. Также у нас Вы можете скачать последние версии веб-сервера Apache, интерпретатора PHP, сервера баз данных MySQL.

Наш портал также тесно связан с проектом TopServer, который предлагает бесплатный сервер приложений, состоящий из веб-сервера Apache, интерпретаторов PHP и PERL, сервера баз данных MySQL.

Мы надеемся, что информация, предоставляемая нами, будет Вам полезна.
